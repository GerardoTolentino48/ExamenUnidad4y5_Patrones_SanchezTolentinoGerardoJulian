Juego de gestión de una ciudad virtual cono los patrones "Object Pool" "MVC" "Singleton" "Bridge" "Memento"

El simulador de juego en formato visual utiliza la estructura lógica de patrones de diseño, esto con el fin de tener una mejor estructuración de código

Este codigo es bastante simple en su ejecución, basta con tener descargados el html y el css dentro de una misma carpeta y en el explorador de archivos dar doble clic sobre el html, esto abrirá una página en chrome con la lógica y estilos CSS funcionando correctamente.
